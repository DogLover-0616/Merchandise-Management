# 物販進捗管理アプリ

個人の物販ビジネス（仕入れ〜発送・到着まで）のステータス管理や利益計算をスムーズに行えるWebアプリケーションです。

## 主な機能
- 商品情報（仕入額・販売額・仕入日・発送日・到着日など）の登録・編集・削除
- ステータス（発送待ち・発送済み・取引終了・キャンセル予定）のワンクリック切り替え
- 合計利益や在庫原価の自動計算・表示
- 登録データのブラウザ内（ローカルストレージ）への自動保存

## 使用技術
- React
- TypeScript
- Vite
- Lucide React (アイコン表示)

## 起動方法

1. 依存関係のインストール
```bash
npm install
```

2. 開発用サーバーの起動
```bash
npm run dev
```

3. ブラウザで [http://localhost:5173](http://localhost:5173) にアクセス
